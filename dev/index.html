Introduction

DecomposingRepresentations.jl is a Julia package that provides an API for decomposing vector space representations of reductive groups together with a concrete implementation of group actions on multivariate polynomials using DynamicPolynomials.jl.

Quick start

julia> using DecomposingRepresentations
       
       # ----- EXAMPLE 1 -----
julia> @polyvar x y z(x, y, z)
julia> vars = [x, y, z]3-element Vector{Variable{Commutative{CreationOrder}, Graded{LexOrder}}}:

 y
 z
julia> SO3 = LieGroup("SO", 3)LieGroup SO(3)

  LieAlgebra 𝖘𝖔(3)
   number type (or field): ComplexF64
   weight type: Int64
   dimension: 3
   rank (dimension of Cartan subalgebra): 1
julia> a = MatrixGroupAction(SO3, [vars])MatrixGroupAction of SO(3)

julia> V = FixedDegreePolynomials(vars, 2)FixedDegreePolynomials space of dimension 6
 variables: x, y, z
 degree: 2
julia> ρ = GroupRepresentation(a, V)GroupRepresentation of SO(3) on the 6-dimensional vector space
 Lie group: SO(3)
julia> irr_decomp = IrreducibleDecomposition(ρ)IrreducibleDecomposition of SO(3)-action on 6-dimensional vector space
 number of irreducibles: 2

julia> sum([dim(irr) for irr in irreducibles(irr_decomp)])6
julia> basis(space(irr_decomp[1]))1-element Vector{Polynomial{Commutative{CreationOrder}, Graded{LexOrder}, ComplexF64}}:

julia> basis(space(irr_decomp[2]))
